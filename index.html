Course Overview

Web Desinging for Beginers

This is a self-paced online course on the fundamentals of how to design websites. You donâ€™t need any prior knowledge or software development experience!

The contents cover the basic building blocks for web designing, including introducing WWW and Internet, essential HTML elements, applying CSS styles, creating interactivity using Javascripts and designing responsive web pages. You can follow this course and gather the technical skills and knowledge required to be a web designer.

The study materials include lecture videos, notes, quizzes and discussion forums. In addition, there are interactive programming playgrounds embedded in the course to practice within this course space. We highly encourage hands-on practice throughout the course. The sections are arranged in a sequential manner. Once you complete a section, you can proceed to the next section. Please note that the passing grade for each quiz has been set to 40%.

This course is designed and made by a team that loves web designing to inspire the next generation of web designers.
